## Descrição do projeto

Esse projeto tem como objetivo mostrar a utilização do Algotitmo do Knapsack DINAMICO COM OPT aplicado a uma ideia de inventário utilizados em alguns jogos RPG. A ideia é utilizar o nosso score sendo:

*(Valor do Item * Raridade) / Peso*

Dando assim uma prioridade para itens de maior raridade. Alem disso, é necessario encaixar o item selecionado em um grid, como nas imagens abaixo:

![Imagem 1](img\9kywwC.png) <br>

![Imagem 2](img\how-did-we-go-from-this-to-this-v0-bp4zbyja7vcc1.jpg)<br>


O programa tambem tem uma opção para fazer um benchmark entre Guloso X Dinamico, e compara o tempo e mostra os itens escolhidos por ambos.
